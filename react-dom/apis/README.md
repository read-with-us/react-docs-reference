# React DOM APIs

> react-dom 패키지에는 웹 애플리케이션(브라우저 DOM 환경에서 실행되는)에서만 지원되는 메서드가 포함되어 있습니다. 이 메서드들은 React Native에서는 지원되지 않습니다.
